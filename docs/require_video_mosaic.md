# 動画固定モザイクツール

## 概要

複数の動画に対して、同じ位置と同じ形状のモザイクを固定で一括適用するツール。
顔、ロゴ、固定表示物など、複数動画で同じ範囲を隠したい用途を想定する。

## 前提

- 処理、プレビュー、エクスポートはブラウザ上で行う。
- 動画を複数追加できる。
- 追加した動画には、同じモザイク設定を固定で適用する。
- 動画ごとの個別モザイク位置調整はMVPでは扱わない。
- モザイク設定は動画の幅と高さに対する正規化座標で保持する。
- GitHub Pages で配信できる静的サイトとして実装する。

## 入力

- 複数動画ファイルをドラッグ&ドロップまたはファイル選択で追加できる。
- 対応形式は `mp4`, `webm`, `mov` を主対象とする。
- 追加した動画は一覧表示する。
- 動画一覧からプレビュー対象の動画を選択できる。
- 動画の解像度、長さ、ファイル名を一覧で確認できる。

## モザイク形状

- モザイク領域は2つの円を基準に設定する。
- 2つの円の外側を、2本の接線と2つの円弧でつないだ領域をモザイク範囲とする。
- 見た目は、2つの円を太さのある帯でつないだカプセル状の領域とする。
- 2つの円は、それぞれ任意の箇所に固定できる。
- 円の中心点をドラッグすると、その円を移動できる。
- 円周をドラッグすると、その円の半径を変更できる。
- 接線と円弧は、2つの円の中心座標と半径から自動計算する。
- 2つの円が重なる、または片方の円がもう片方を内包する場合でも、破綻しない代替形状として2円の合成領域を表示する。

## 座標仕様

- 円の中心座標と半径は、動画サイズに対する正規化値で保持する。
- 中心座標は `0` から `1` の範囲を基本とする。
- 半径は動画の長辺に対する比率で保持する。
- エクスポート時は、各動画の実ピクセルサイズへ変換して適用する。
- 解像度が異なる動画でも、同じ相対位置にモザイクを適用する。
- 縦横比が大きく異なる動画が混在する場合は、プレビューと一覧に警告を表示する。

設定データ例:

```json
{
  "version": 1,
  "shape": "twoCircleTangent",
  "circleA": { "cx": 0.35, "cy": 0.42, "r": 0.06 },
  "circleB": { "cx": 0.58, "cy": 0.44, "r": 0.07 }
}
```

## ピクセルモザイク

- モザイク方式はピクセルモザイクとする。
- モザイクのピクセルサイズは、対象動画の長辺の `1/100` とする。
- ピクセルサイズは動画ごとに自動算出する。
- ユーザーがモザイク強度を選択するUIはMVPでは用意しない。
- プレビューとエクスポートでは同じ算出ルールを使う。

計算式:

```ts
const pixelSize = Math.max(1, Math.round(Math.max(videoWidth, videoHeight) / 100));
```

計算例:

- `1920x1080` の動画は、長辺 `1920` からピクセルサイズ `19` とする。
- `1080x1920` の動画は、長辺 `1920` からピクセルサイズ `19` とする。
- `1280x720` の動画は、長辺 `1280` からピクセルサイズ `13` とする。

## プレビュー

- 選択中の動画をプレビューできる。
- プレビューでは再生、停止、シークができる。
- プレビュー上にモザイク結果を表示する。
- 編集用ガイド表示のオン、オフを切り替えできる。
- ガイド表示オンでは、2つの円、中心点、円周ハンドル、接線、円弧、モザイク範囲の輪郭を表示する。
- ガイド表示オフでは、実際の出力に近いモザイク結果のみを表示する。
- プレビュー対象動画を切り替えても、モザイク設定は共通のまま維持する。

## UI

- 動画追加エリア
- 動画一覧
- プレビュー
- 再生UI
- モザイク編集キャンバス
- ガイド表示切り替え
- モザイク設定リセット
- 全動画への一括適用ボタン
- エクスポート進捗表示
- キャンセルボタン
- 処理ログ表示

## 操作仕様

- 動画を追加すると、先頭の動画を自動的にプレビュー対象にする。
- 動画一覧の項目をクリックすると、プレビュー対象を切り替える。
- 円の中心点ドラッグで円を移動する。
- 円周ドラッグで円の半径を変更する。
- 円の移動と半径変更は、プレビュー動画上の座標から正規化座標へ変換して保持する。
- モザイク設定リセットで初期位置に戻せる。
- エクスポート中も進捗とログを表示する。
- エクスポート中はユーザーがキャンセルできる。
- ユーザーキャンセルは正常な中断結果として扱い、エラー扱いしない。
- 自動的な最大リトライ回数や自動停止条件は設けない。

## エクスポート

- 追加された全動画に同じモザイク設定を適用して出力する。
- 出力ファイルは入力動画ごとに個別に生成する。
- 出力ファイル名は元ファイル名へ `_mosaic` を付ける。
- 例: `sample.mp4` は `sample_mosaic.mp4` として出力する。
- まずは個別ダウンロードをMVPとする。
- ZIP一括ダウンロードは後続拡張とする。

## 後続拡張

- モザイク設定JSONの保存と読み込み。
- 動画ごとの個別位置補正。
- 複数モザイク領域。
- ZIP一括ダウンロード。
- キーフレームによるモザイク位置移動。
