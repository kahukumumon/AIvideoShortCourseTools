# 動画編集ツール

## 概要

ブラウザ上で完結する動画編集ツールのMVP仕様。
複数の無音動画と複数音声を組み合わせて、1つの音声付き動画として `mp4` で出力する。

## 前提

- 編集、再生、エクスポートはブラウザ上で行う。
- 動画素材は無音動画のみ対応とする。
- プロジェクト状態はブラウザリロードで消える。
- 動画データをブラウザストレージへ保持しない。
- GitHub Pages で配信できる静的サイトとして実装する。

## 実装方針

- フロントエンドは `React` を採用する。
- ビルド基盤は `Vite + React` を前提とする。
- GitHub Pages へ静的ファイルとしてデプロイできる構成にする。
- アプリ本体はクライアントサイドで動作する。
- `SSG` は静的配信のために利用し、編集処理自体はクライアントサイドで扱う。
- 開発用のソースコードは `tools/video_edit` とは別ディレクトリで管理する。
- 配信用のビルド成果物を `tools/video_edit` 配下に配置する。
- `tools/video_edit` は GitHub Pages に配置する静的ファイル置き場として扱う。

## 出来ること

- 動画のカット
- 複数動画ファイルの追加
- 動画タイムライン内での動画クリップ並び替え
- 動画タイムライン内での動画クリップ結合
- 音声タイムライン内での音声クリップ結合
- 動画タイムラインと音声タイムライン群を統合した動画出力
- 動画クリップの再生速度変更
- 音声クリップの再生速度変更
- 動画クリップのフェードイン、フェードアウト
- 音声クリップのフェードイン、フェードアウト
- 音声クリップごとの音量調整
- 音声トラック内の複数クリップへの一括音量調整
- 音声クリップごとの `10ms` 間隔 `RMS` グラフ表示
- モザイクタイムラインの追加
- モザイククリップごとの楕円形モザイク設定
- ピクセルモザイク方式の適用

## タイムライン構成

- 動画トラックは複数クリップを配置できる。
- 音声トラックは複数本持てる。
- モザイクトラックは複数本持てる。
- タイムラインの先頭時刻は `0:00.0` とする。
- 動画クリップはそれぞれ独立した開始時刻を持てる。
- 動画トラック内ではクリップの時間重なりを禁止する。
- 動画トラック内では空白区間を許可する。
- 同一音声トラック内ではクリップの時間重なりを禁止する。
- 別音声トラック間ではクリップの時間重なりを許可する。
- 同一モザイクトラック内ではクリップの時間重なりを禁止する。
- 別モザイクトラック間ではクリップの時間重なりを許可する。
- 重なった音声はエクスポート時にミックスする。

## UI

- 動画プレビュー

- 音声ドロップ領域

- モザイクタイムライン

- 再生UI（再生、停止）

## タイムライン操作

- クリップをドラッグして開始位置を変更できる。
- クリップをドラッグして同一トラック内の前後順を入れ替えできる。
- クリップの端を `Shift + ドラッグ` してトリミングできる。
- クリップの端をドラッグして再生速度を変更できる。
- クリップを右クリックしてフェードイン、フェードアウトを設定できる。
- クリップを右クリックして削除できる。
- クリップを右クリックして分割できる。
- クリップを右クリックして結合できる。
- モザイククリップをドラッグして開始位置を変更できる。
- モザイククリップの位置とサイズをプレビュー上でドラッグ操作できる。
- モザイク楕円をプレビュー上で回転できる。

## 操作仕様

### 分割

- 分割位置は再生ヘッド位置とする。

### 配置

- 動画クリップ、音声クリップともに同一トラック内で重複配置を禁止する。
- モザイククリップは同一モザイクトラック内で重複配置を禁止する。
- 動画クリップは同一動画ファイル由来でも独立したクリップとして扱う。
- 動画トラックでは空白区間を維持したまま配置変更できる。
- クリップ移動時は移動対象自身を除いた空き区間を探索して再配置する。
- モザイクトラックは空白区間を維持したまま配置変更できる。

### モザイク

- モザイクは専用のモザイクトラック上のモザイククリップとして管理する。
- モザイクトラックはユーザーが必要に応じて追加できる。
- モザイククリップは開始時刻と終了時刻を持つ。
- モザイク方式はピクセルモザイクとする。
- モザイクのピクセルサイズは対象動画の長辺の `1/100` とする。
- モザイククリップはプレビュー上で楕円形マスクとして表示する。
- モザイクの適用位置はプレビュー上のドラッグで変更できる。
- モザイクのサイズはプレビュー上のドラッグで変更できる。
- モザイク楕円は回転角度を持ち、プレビュー上で回転操作できる。
- モザイク位置、サイズ、回転はモザイククリップごとに保持する。
- プレビューとエクスポートでは同一のピクセルサイズ算出ルールを使う。
- 同時刻に複数モザイククリップが存在する場合、それぞれ独立して適用する。

### 結合

- 結合対象は同一トラック内で隣接している同種クリップのみとする。
- 動画クリップは動画トラック内でのみ結合できる。
- 音声クリップは音声トラック内でのみ結合できる。
- 結合結果は1つの新しいクリップとして扱う。
- 結合前の個別クリップ属性は保持しない。
- 結合時点の編集結果をマージした新規クリップを生成する。
- 速度変更、フェード、音量は結合結果に反映済みの素材として扱う。
- 結合後は個別編集履歴を保持しない。
- 各パラメータは、どちらか一方が標準値であれば、もう一方の変更値を適用する。
- 両方とも変更値であれば、時間的に早いほうの値を採用する。

### 再生速度

- 動画クリップ、音声クリップともに再生速度変更を可能とする。
- 再生速度の範囲は `0.25x` から `4x` とする。
- 再生速度変更時はピッチを維持する。
- 再生速度変更に応じて、タイムライン上のクリップ長を伸縮させる。

### フェード

- 動画クリップのフェードは黒フェードとする。
- 音声クリップのフェードは音量フェードとする。
- フェード時間は秒数指定とする。
- フェード時間は `0` からクリップ長の `50%` まで指定できる。

### 音量

- 音声クリップごとに音量を `dB` で設定できる。
- 音声トラック単位の永続的な音量属性は持たない。
- 音声トラック内の複数クリップに対して、一括で `dB` 値を適用できる。
- 音量の範囲は `-60dB` から `+12dB` とする。

### RMS グラフ

- 音声クリップごとに `10ms` 間隔で `RMS` 値を計算して保持する。
- `RMS` は `-80dB` から `0dB` の範囲を視覚化できる値として扱う。
- `RMS` グラフは音声クリップ背景に半透明で重ねて表示する。

## 使い方

1. 無音動画を動画ドロップ領域にドラッグ&ドロップする。
2. 複数動画を一度に追加するか、複数回に分けて追加する。
3. 追加した動画は動画タイムライン上の独立クリップとして配置される。
4. 最初の動画の第1フレームをプレビューに表示する。
5. 動画クリップを編集する。
6. 音声を音声ドロップ領域にドラッグ&ドロップする。
7. 音声クリップを音声タイムラインに追加する。
8. 音声クリップを編集し、`RMS` グラフで音量ピークを確認する。
9. 必要に応じてモザイクトラックを追加し、モザイククリップを配置する。
10. プレビュー上でモザイクの位置、サイズ、回転を調整する。
11. 動画と音声を共通シークバーで同時再生して確認する。
12. 動画トラックと音声トラック群とモザイクトラック群を統合してエクスポートする。

## エクスポート

- 出力形式は `mp4` のみとする。
- 動画コーデックは `H.264` とする。
- 音声コーデックは `AAC` とする。
- 複数動画クリップは時系列順に連結して1本の映像ストリームを生成する。
- 各動画クリップにはトリミング、速度変更、フェードを反映してから連結する。
- モザイククリップの時間範囲と形状設定を動画へ反映してから出力する。
- モザイクのピクセルサイズは出力動画の長辺の `1/100` を基準に適用する。
- 複数音声トラックの音声は1本にミックスして動画へ埋め込む。
- 複数動画ファイルを扱う場合、`FPS` は最初の動画を基準とする。
- 解像度は直前または先頭動画に合わせて統一し、必要に応じてリサイズする。
